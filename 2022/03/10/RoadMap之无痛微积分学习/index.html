RoadMap之无痛微积分学习

现在状态：

  1. 学过微积分，但学得不是很好
  2. 在学习工作中不是经常使用，但还是有用到的概率

期望状态：

  1. 达到高级新手水平
  2. 满足学习工作中的应用需求
  • 对微分方程使用极限定义和微分规则
  • 使用渐近线，关键点画函数草图，使用微分测试函数的增减性，凸凹性
  • 使用微分求解应用最大最小问题
  • 使用微分求解related rate 问题
  • 使用黎曼和和积分基础理论估计积分
  • 使用高级积分技巧算积分，such as inverse substitution, partial fractions and integration by parts.
  • 使用 L hosptial rule 来evaluate certain indefinite forms
  • 确定不定积分的收敛性，计算不定积分的convergent
  • 找到函数一个点附近的泰勒级数展开

困难与疑虑：

  1. 不知道该如何学
  2. 希望有系统性框架，但由于不是十分需要，又不想花费过多时间
  3. 害怕微分方程

灵感与心得

  1. 我悟了，我对大雾的心有余悸来源于微积分没学好 🤦‍♀️

differential operator 符号▼ 名字叫del，也叫nabla

inifessimal vector

curl del 叉乘 u

curl of gradient is vector 0

divergence of curl is scalar 0

怎么区分divergence 和 gradient ？ differential operator + dot product

涡旋 vorticity 二维速度场的del 叉乘 出来三维的矢量

laplacian del 点乘自己 二阶偏微分之和 标量场和矢量场不一样

pde中很有用

laplace function

wave function

diffusion function
