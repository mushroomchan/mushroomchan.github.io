安装主题

  1. 在博客目录执行命令

    通过 npm 安装的话 ～/themes/ 下不会有新文件产生。还有另一种安装方法是下载 release 版本代码，未尝试。

  2. 在博客目录下创建/_config.fluid.yml，将github仓库中_config.yml内容复制进去。

  3. 在~/_config.yml中的主题修改了fluid

  4. 创建[关于页]

涉及配置文件

  1. “博客配置“ 指的 Hexo 博客目录下的 _config.yml，”主题配置“ 指的是 theme/fluid/_config.yml 或者 _config.fluid.yml
  2. 每次无论 hexo g 或 hexo s，都最好先使用 hexo clean 清除本地缓存

网页统计

使用百度的，用百度账号开通

网络域名和网站首页填写 username.github.io 或者你的域名，点链接可以跳转到网站就是ok了

注册链接

Homepage URL & Authorization callback URL 填写 http://username.github.io ，注意要带上https://，否则会提示错误URL

⚠️注意：是 https，不要漏掉s，否则验证的时候会重定向回主页，无法正确初始化

注册好后跳转到对应页面，生成一个clien secret，获得clientId & clientSecret

在主题配置中修改找到 comments 和 gittalk 段，修改相关信息
